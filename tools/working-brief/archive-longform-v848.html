6 · Standing laws — condensed

Data is the source of truth — engine locates, LLM voices; accuracy outranks everything, including UX and cosmetics, in every triage.

Fix the source, not the symptom — name the root in one sentence, fix the CLASS.

A control that can FAIL is worth more than a test that passes. This run's proof is sharper than the last one's: the ink solver had a test written for exactly its bug, and the test could not catch it — a loose ceiling on a convenient fixture. Every control now carries a denominator, and where it matters I revert the source and watch the assertion fail before I believe it.

The probe is wrong before the code is, more often than you would think. Three fixtures and one scan lied to me this run. When a probe reports something surprising, suspect the probe first.

An offset is not a boundary. Deleting code by "start marker → next export" silently took two live symbols with it. Cut by named boundaries.

A colour helper is only correct relative to a GROUND — and the ground is not always the card.

Never ship a visual you have not SEEN. The rail is sitting in v813 unconfirmed for exactly this reason, flagged rather than claimed.

Re-audit your own run — and question the agents. Their substance was good and their citations were not; the two biggest engine defects of this run were found by none of them.
