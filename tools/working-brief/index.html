The productivity Planner — neurodivergent-UX, greenlit 17 July

Evidence-based, from the market deck: neuroinclusive is universal design — the same choices that unblock an ADHD user unblock every tired, distracted, overloaded one. None of the five leaders (Notion, Todoist, TickTick, ClickUp, Motion) ship #1, #3 or #9 — open ground. These drop onto the Planner that already exists.

  1. One task / Focus Mode — hide all but the current action, one primary CTA. The app leans this way via soft-open + collapse; a dedicated mode is unbuilt.
  2. AI task-decomposition on capture — break a task into 3–7 micro-steps, first pre-selected; kills initiation paralysis at the moment it strikes. Shipped v710 per the record — confirm on a live look.
  3. Gentle overdue — overdue fades to neutral + one-tap "roll everything forward"; never a stack of red badges (shame → avoidance → churn). Shipped v708 per the record — confirm on a live look.
  4. Sensory control panel — the user controls animation, sound, colour-intensity, density; no autoplay, no unprompted motion. Needs your UX spec.
  5. Visual time-anchoring — every task shows an estimated duration; a shrinking-disc timer (not numerals) for the active one. Time-blindness accommodation, Motion's whole moat. Needs a duration schema script — yours to run.
  6. Zero-friction capture — voice-first, natural-language parse, capture from lock screen / share sheet, under two seconds. Never paywall capture.
  7. Save-and-resume by default — every flow pausable and resumable with visible progress and a one-line "you were here" re-orient.
  8. Predictable structure, opt-in novelty — navigation never moves; redesigns ship as opt-in themes (the hard no-jarring-redesign rule).
  9. Body-doubling & forgiving rewards — optional co-working presence; streaks that forgive gaps rather than reset to zero.
  10. Pomodoro focus sessions — your addition. A 25-min shrinking disc on a task → rest bell → 5–10 min break (a mini rest-gate, never guilt) → a different return bell. The two bells are your art. Parked; shares the duration script with #5. Trigger: "go pomodoro".
