The five release gates — every release passes all five

David's framework (2026-07-22). Eighteen audits under five gates — full spec verbatim in RELEASE_GATES.md. Run the Deterministic Astrology Audit before anything else.

1Astrological Fidelity

Is the astrology correct and methodologically consistent? Every conclusion deterministic, every "why" traceable to engine output; no duplicate computations, no conflicting rulings, no silent fallback changing method.

2Narrative Fidelity

Does the reading tell ONE coherent story without the LLM inventing meaning? One governing idea, one protagonist, one arc; strong ending; cut true-but-unnecessary facts. (Full contract: NARRATIVE_AUDIT.md.)

3Engineering Integrity

Is every engine output deterministic, traceable, and free of ambiguity? Inspect every JSON before the LLM (no nulls, no duplicates, no ambiguous subjects, no unused fields). No magic — every behaviour explainable.

4Visual & UX Excellence

Does every interaction feel polished, intuitive, and unmistakably Velea? IA, onboarding, delight, accessibility, performance, a mathematically consistent design system.

5Business Excellence

Does it maximize trust, retention, and conversion with no friction and no money leaks? Conversion, retention, error recovery, and every business rule documented — nothing exists "because the code does it."
